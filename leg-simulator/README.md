# LEG-Simulator

Real-time visualization of electrical energy flows within a Local Energy Grid (LEG).

## Quick Start

```bash
cd leg-simulator
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open http://localhost:8050
